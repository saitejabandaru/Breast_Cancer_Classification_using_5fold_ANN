Breast Cancer Prediction System

A machine learning web application that predicts whether a breast tumor is Benign or Malignant using diagnostic tumor features.

This system is mainly intended for researchers, students, and medical data analysts, because the model requires detailed tumor feature values such as radius, texture, perimeter, area, smoothness, compactness, concavity, and related diagnostic measurements. A normal person usually will not have direct access to all these values without a medical dataset or diagnostic report.
